Neovim Cheatsheet

Leader key: Space

Window Splits

Ctrl + w vVertical split
Ctrl + w sHorizontal split
Ctrl + w qClose current window
Ctrl + w oClose all other windows
Ctrl + w =Equal window sizes
Ctrl + w _Maximize height
Ctrl + w |Maximize width
:vsp fileVertical split with file
:sp fileHorizontal split with file

Window Navigation

Ctrl + hMove to left window
Ctrl + jMove to bottom window
Ctrl + kMove to top window
Ctrl + lMove to right window
Ctrl + w wCycle through windows
Ctrl + w HMove window to far left
Ctrl + w JMove window to bottom
Ctrl + w KMove window to top
Ctrl + w LMove window to far right
Ctrl + w rRotate windows

Tab Management

:tabnewOpen new tab
:tabnew fileOpen file in new tab
:tabcClose current tab
:taboClose all other tabs
gtNext tab
gTPrevious tab
3gtGo to tab 3
:tabm 0Move tab to first position
:tabm +1Move tab right
:tabsList all tabs

Buffer Navigation

TabNext buffer (NvChad)
Shift + TabPrevious buffer (NvChad)
Space + xClose buffer (NvChad)
:bnNext buffer
:bpPrevious buffer
:bdDelete buffer
:lsList all buffers
:b nameSwitch to buffer by name
:b3Switch to buffer 3

Jump & Search

?patternSearch backward
